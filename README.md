# SublimeCobra #

## Overview ##

My [Cobra][cobra] settings for [Sublime Text][st].

Based on BOOMER's [original work][wiki] from the Cobra Wiki.

[cobra]: http://cobra-language.com
[st]: http://www.sublimetext.com
[wiki]: http://cobra-language.com/trac/cobra/wiki/SublimeText

## Installing ##

Within Sublime Text, click `Preferences > Browse Packages...` and create a folder named Cobra. Clone this repo into the folder. Package Manager support will hopefully come soon.
